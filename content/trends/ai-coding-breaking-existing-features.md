## 这是什么  
AI Coding Breaking Existing Features 指在AI辅助编程过程中，由AI生成的代码无意中破坏已有正常功能的现象。这不仅是偶发错误，而是独立开发者普遍反馈的痛点，尤其在代码迭代或重构时风险更高。

## 谁应该关注  
使用AI编码工具（如GitHub Copilot、Cursor等）的独立开发者及小型团队最需追踪此趋势，因其缺乏冗余测试资源。产品经理也应关注，避免因AI生成的代码破坏功能导致用户信任流失。创业者需评估AI编码在关键业务中的风险边界。
