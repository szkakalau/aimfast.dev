## What is it（这是什么）
Jelly UI 是一种交互设计概念，为原生 HTML 表单控件（如按钮、滑块、复选框）注入软体物理效果。它模拟果冻般的弹性形变与回弹动画，使静态控件在点击、拖拽时产生生动的触感反馈。对独立开发者而言，Jelly UI 提供了一种低成本提升界面趣味性和用户情感连接的新思路。

## Why now（为什么现在出现）
Jelly UI 的出现源于三个因素：首先，Web 前端性能提升（如 CSS Houdini、WebGPU）使得在浏览器中实时模拟软体物理成为可能；其次，用户对千篇一律的扁平化设计产生审美疲劳，渴望更具生命力的交互体验；最后，独立开发者社区追求差异化产品，Jelly UI 能以极低开发成本在表单等基础组件上建立独特品牌记忆点。

## Who's behind it（谁在推动）
目前 Jelly UI 主要由 Hacker News 社区中的独立开发者或小型设计实验团队推动，尚未出现大型公司或知名开源项目主导。其源头可能来自个别前端开发者将物理引擎（如 Matter.js）与表单控件结合的个人实验项目。由于信源仅 hn 且提及次数为 1，当前缺乏明确的组织或个人背书，更像是一个萌芽中的社区概念。

## Market signals（市场信号）
Jelly UI 当前处于 nascent（萌芽）阶段，趋势分数 48/100，信源仅 1 个（hn），总提及 1 次。这表明它尚未在主流技术社区形成讨论热度，跨平台提及模式为零。信号强度极弱，但分数接近 50 暗示潜在关注度可能随首个演示视频或开源库发布而快速上升。

## Commercial opportunities（商业化机会）
1. **Jelly Form Builder**：开发一个拖拽式表单构建工具，内置 Jelly UI 软体物理效果，让无代码用户快速创建趣味交互表单。
2. **UI 动画组件库**：打包 10-15 个常用表单控件的 Jelly 风格动画组件（React/Vue），以付费插件形式售卖。
3. **品牌定制服务**：为中小型 SaaS 产品提供表单交互升级咨询，通过 Jelly UI 提升用户注册转化率。

## Related terms（相关趋势）
- **Neumorphism**（新拟物风格）：与 Jelly UI 同属“超越扁平化”的设计运动，但 Neumorphism 侧重视觉质感，Jelly UI 侧重物理动态。
- **Micro-interactions**（微交互）：Jelly UI 是微交互的一个具体实现方向，强调反馈的生动性与拟物感。
- **CSS Houdini**：Jelly UI 的物理渲染可能依赖 Houdini API 实现高性能自定义动画，两者在技术栈上紧密相关。

## Product ideas（产品创意）
1. **JellyKit**  
   一个轻量级 JavaScript 库，为任何 HTML 表单控件一键添加 Jelly 物理效果。  
   **时机**：nascent 阶段缺乏标准化工具，首个开源库能快速吸引早期采用者，形成社区基础。

2. **FormJelly**  
   面向独立开发者的 Figma 插件，用于在原型设计中预览 Jelly UI 交互，并自动导出 CSS/React 代码。  
   **时机**：设计到代码的转化是痛点，目前无竞品，且 Figma 插件生态成熟，开发成本低。

3. **JellyMetrics**  
   一个 A/B 测试工具，专门测量 Jelly UI 对表单转化率、用户停留时长的影响。  
   **时机**：独立开发者需要数据证明 Jelly UI 的商业价值，工具可填补“效果量化”空白，为后续商业化提供背书。
