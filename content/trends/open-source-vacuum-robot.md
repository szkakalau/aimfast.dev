## What is it（这是什么）
Open-Source Vacuum Robot 指将扫地机器人的硬件设计、软件代码和组装指南完全公开的项目，例如 makerspet/oomwoo。这意味着开发者可以自由下载、修改和制造自己的扫地机器人，无需依赖商业品牌。它降低了消费级机器人入门门槛，让独立开发者能像组装电脑一样定制家庭清洁设备。

## Why now（为什么现在出现）
Open-Source Vacuum Robot 的出现源于三个驱动因素：首先，开源硬件生态成熟，如3D打印和低成本微控制器普及，使个人制造复杂机器人成为可能；其次，消费者对数据隐私和功能定制的需求上升，商业扫地机器人封闭系统引发不满；最后，makerspet/oomwoo 项目在 GitHub 上获得高星，证明了社区对透明、可扩展的家庭机器人方案的渴求，推动该术语在 2026 年 7 月首次被追踪到。

## Who's behind it（谁在推动）
目前主要推动者是 makerspet 组织及其项目 oomwoo，这是一个在 GitHub 上获得高星的开源社区。项目贡献者包括硬件工程师、嵌入式开发者与机器人爱好者，他们通过共享代码、电路图和3D模型协作。暂无大型商业公司直接参与，但该趋势可能吸引如 Arduino、Raspberry Pi 等开源生态平台后续支持。
