## What is it

Jelly UI is a design technique that applies soft-body physics to native HTML form controls. Instead of static, rigid buttons and sliders, elements behave like gelatin—they squish, wobble, and bounce when users interact with them. Think of a submit button that compresses under your cursor and then springs back into shape, or a range slider that ripples like water when dragged. For indie developers, Jelly UI represents a way to inject personality and tactile feedback into web apps without abandoning standard HTML semantics. It’s not a library or framework yet, but a conceptual approach that blends frontend animation with physics simulation. The core idea is that digital interfaces feel more alive and responsive when they mimic real-world material properties, making interactions more intuitive and delightful.

## Why now

Jelly UI is emerging now for several reasons. First, browser APIs for animation and physics have matured dramatically. CSS custom properties, Web Animations API, and Canvas-based physics engines are now broadly supported, making soft-body effects feasible without heavy dependencies. Second, users are growing bored with flat, sterile interfaces. After years of minimalism, there’s a hunger for playful, emotionally resonant design—especially in consumer apps. Third, indie developers are increasingly competing on UX polish rather than features. A subtle squish effect can differentiate your product from a dozen competitors. Finally, the indie hacker community on Hacker News and Product Hunt actively rewards novel interaction patterns. Jelly UI taps into this appetite for experimentation, offering a low-cost way to stand out.

## Related terms

**Micro-interactions** is the closest cousin. Both focus on tiny, delightful feedback loops in UI, but Jelly UI specifically applies physics rather than pre-canned animations. **Physics-based UX** is a broader trend seen in game UI and mobile apps (e.g., Apple’s spring animations). Jelly UI brings this to the web. **Skeuomorphism 2.0** is another related term. Unlike the old leather-and-wood aesthetic, this new wave uses physics to mimic material behavior without literal textures. Together, these trends suggest a shift toward interfaces that feel alive rather than flat.
