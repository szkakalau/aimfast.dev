## What is it（这是什么）

Nectar 是一个新兴的开源框架，它结合了 Rust 语言的安全性和 React 的组件化开发模式，并能将代码编译为 WebAssembly（Wasm）运行。简单来说，它让开发者用类似 Rust 的语法构建高性能、类型安全的前端应用，最终在浏览器中以接近原生的速度执行，适合追求极致性能和可靠性的独立开发者。

## Why now（为什么现在出现）

随着 WebAssembly 在浏览器中的普及（如 Wasm GC 和组件模型成熟），开发者对高性能前端框架的需求增加。同时，Rust 因其内存安全和并发特性，在系统编程领域备受青睐。Nectar 的出现正是为了弥合这两者之间的鸿沟：将 Rust 的严谨带入前端开发，解决 JavaScript 在大型应用中的性能瓶颈和类型安全问题，满足独立开发者对“更安全、更快速”工具的渴望。
