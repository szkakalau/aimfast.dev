## What is it

hello-agents is a tutorial project that teaches indie developers how to build AI agents from scratch. Think of it as a hands-on workshop in code form—it walks you through the core principles of agent architecture, then shows you the practical implementation. Unlike using a black-box framework, hello-agents strips away the abstraction so you understand every moving part: how agents perceive environments, make decisions, and take actions. For indie hackers, this means you can customize agents for your specific SaaS use cases without being locked into someone else's opinionated library. It's a learning resource, not a production tool, but the knowledge is immediately applicable.
