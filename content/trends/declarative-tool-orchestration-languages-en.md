## What is it  
Declarative Tool Orchestration Languages are a nascent category of sandboxed programming languages, such as Skillscript, designed to let AI agents invoke tools in a safe, predictable, and declarative manner. Unlike imperative scripting, these languages specify *what* outcome is desired rather than *how* to achieve it, reducing the risk of unintended side effects. This approach complements the growing agent skills ecosystem by offering a structured layer for tool orchestration that prioritizes security and determinism.
