## 1. What is it（这是什么）
Java 是一种跨平台的面向对象编程语言，由 Sun Microsystems 于 1995 年推出，现由 Oracle 维护。它采用“一次编写，到处运行”的理念，通过 Java 虚拟机（JVM）在 Windows、Linux、macOS 等系统上运行。对独立开发者而言，Java 是企业级后端开发、Android 应用和大型系统的首选语言之一，学习曲线中等，但生态极其成熟。
