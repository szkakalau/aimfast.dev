## 1. What is it（这是什么）

Tailwind CSS 是一个“实用优先”的 CSS 框架，让你通过组合预定义的类（如 `flex`、`text-center`）直接在 HTML 中构建自定义 UI，无需写一行自定义 CSS。对独立开发者来说，它大幅缩短了样式编写时间，同时保持设计一致性和响应式能力。

## 2. Why now（为什么现在）

虽然 Tailwind CSS 已存在多年，但近期在独立开发者社区中重新爆发关注。原因有三：一是前端项目复杂度上升，开发者追求更快的原型迭代；二是 AI 辅助编码工具（如 Copilot）与 Tailwind 的类名系统高度契合，能显著提升生成效率；三是 Next.js、Remix 等现代框架默认推荐 Tailwind，形成生态锁定效应。当前阶段（Nascent）表明这波关注刚起步，存在早期套利窗口。

## 4. Who's Behind It（谁在推动）

核心推动者是 Tailwind Labs 公司及其创始人 Adam Wathan。此外，Vercel（Next.js 开发商）和 Netlify 等基础设施平台在默认模板中内置 Tailwind，加速了采用。独立开发者社区（如 w2solo、掘金）是当前传播的主力，因为 Tailwind 的低心智负担特性完美匹配单人开发者的效率需求。
