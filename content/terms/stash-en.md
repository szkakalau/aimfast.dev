## 9. Product Ideas（产品创意）
- **StashPad**：一款 VS Code 插件，允许开发者用快捷键“暂存”当前选中代码，自动生成标签和上下文描述。现在做是因为 AI 辅助编码导致代码碎片化严重，急需轻量级管理工具。
- **StashBoard**：Web 端仪表盘，聚合 GitHub Gist、CodePen 和本地收藏，支持一键导出为 Markdown 或 JSON。现在做是因为跨平台代码资源分散，统一入口有刚需。
- **StashBot**：Telegram/Discord 机器人，接收用户发送的代码片段并自动分类存储，支持自然语言搜索。现在做是因为聊天工具成为开发者日常协作中心，低门槛接入可快速获客。
