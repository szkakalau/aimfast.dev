## 1. What is it（这是什么）
Tailwind CSS 是一个“实用优先”的 CSS 框架。它不提供预设计的组件（如按钮、卡片），而是提供大量原子化的 CSS 类（如 `text-center`、`bg-blue-500`），开发者直接在 HTML 中组合这些类来构建界面。这就像用乐高积木代替预制模型，让你能快速、一致地定制设计，无需写自定义 CSS。

## 2. Why now（为什么现在）
Tailwind CSS 并非全新事物，但近期在 2026 年 7 月 23-24 日于 w2solo 和掘金社区被独立提及，可能因以下原因重新获得关注：1）独立开发者社区对“快速原型”需求的持续增长，Tailwind 能显著缩短从设计到代码的周期；2）其生态（如 Headless UI）成熟，解决了早期“难以复用”的痛点；3）AI 代码生成工具（如 GitHub Copilot）能高效生成 Tailwind 类名，形成技术协同，降低使用门槛。
