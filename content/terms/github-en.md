## 1. What is it（这是什么）

## 9. Product Ideas（产品创意）

1. **RepoLens**：一个 Chrome 扩展，在 GitHub 仓库页面侧边栏显示代码质量评分、依赖健康度和贡献者活跃度热力图。为什么现在做：开发者越来越依赖 GitHub 做技术选型，但缺乏直观的评估工具。

3. **MergeMate**：智能合并冲突解决工具，基于 AI 自动分析冲突上下文并建议合并方案。为什么现在做：大型开源项目合并冲突频繁，AI 能力已足够成熟来处理此类问题。
