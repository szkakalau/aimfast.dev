## 1. What is it（这是什么）
ATProto 是一个去中心化社交网络协议，由 Bluesky 团队开发。它允许用户在不同服务商之间自由迁移身份和数据，就像电子邮件一样。对独立开发者来说，这意味着可以基于开放协议构建社交应用，而不必受制于单一平台（如 Twitter 或 Mastodon）的 API 限制。

## 2. Why now（为什么现在）
当前社交网络平台（如 Twitter/X）频繁变更 API 政策、限制第三方应用，开发者对“平台绑定”的厌倦达到顶峰。同时，Mastodon 等去中心化方案虽已存在，但用户体验和开发者工具仍不成熟。ATProto 由 Bluesky 团队（前 Twitter 创始人团队）推动，具备更强的技术背书和资本支持。它出现在开发者急需一个“既开放又易用”的社交基础设施的窗口期。
