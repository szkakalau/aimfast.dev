## 1. What is it（这是什么）
Chat SDK 是一套软件开发工具包，让开发者能快速将聊天功能集成到自己的应用或网站中。它通常包含预建的聊天 UI 组件、消息处理 API、实时通信协议，以及用户管理、消息历史等后端服务。独立开发者只需几行代码，就能为自己的产品添加类似 Slack 或 Discord 的聊天体验，而无需从零搭建实时通信基础设施。

## 2. Why now（为什么现在）
当前 AI 应用爆发，用户对“对话式交互”的预期已从聊天机器人扩展到协作工具、客服系统、社交应用等场景。传统自建聊天功能成本高、维护复杂，而现有 SDK 多面向大企业，定价昂贵、集成笨重。同时，Vercel 等平台开始关注实时体验，推动轻量级、可嵌入的 Chat SDK 需求。独立开发者需要一种低成本、开箱即用的方案，来快速验证产品中的社交或协作功能。
