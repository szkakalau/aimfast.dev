## 1. What is it（这是什么）
Next.js 是一个基于 React 的 Web 应用框架，能让开发者轻松构建服务端渲染（SSR）和静态网站（SSG）的网页。它解决了传统 React 应用 SEO 差、首屏加载慢的问题，并提供文件路由、API 路由等开箱即用的功能。简单说，它让 React 网站更快、对搜索引擎更友好。

## 2. Why now（为什么现在）
当前 Web 开发对性能和 SEO 要求越来越高，而 React 单页应用（SPA）的 SEO 问题长期存在。Next.js 通过提供混合渲染模式（SSR + SSG）和增量静态生成（ISR）等技术，在 2026 年成为解决这些痛点的成熟方案。同时，Vercel 等云平台的崛起降低了部署门槛，使得独立开发者能快速使用。
