## 1. What is it（这是什么）
Next.js 是一个基于 React 的全栈 Web 开发框架，由 Vercel 公司维护。它让你可以用 React 组件轻松构建服务端渲染（SSR）和静态生成（SSG）的网站，同时支持 API 路由、文件系统路由和自动代码分割。简单说，它是一个能帮你快速搭建高性能、SEO 友好的现代 Web 应用的“瑞士军刀”。

## 2. Why now（为什么现在）
虽然 Next.js 已存在多年，但近期（2026年7月）在 OSChina 和 DEV Community 上出现的新提及，可能源于两个关键变化：一是 React Server Components 的稳定化，让 Next.js 的混合渲染能力更强大；二是 AI 驱动的低代码工具兴起，开发者需要更高效的框架来快速构建原型。此外，独立开发者对“全栈单仓库”方案的需求持续增长，Next.js 正好填补了这一空白。这个时间点，技术成熟度与市场需求形成共振。
