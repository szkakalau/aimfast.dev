- **V1**: Google Form + GitHub Gist delivery — zero backend code

### Fastest Validation Path (Doable Today)

## 🔍 Opportunity Discovery

### Solo-Founder Product Launches

#### Signal: "Are You in the Weights?" — Model Dataset Query Tool

- **Plain English**: A website where you enter a model name to check if it's in known training datasets. 241 comments show developers are anxious about "who's using my model."
- **Key Judgment**: One-shot query tool with no repeat purchase scenario. Not worth building unless turned into an enterprise monitoring service ($29/month to track if your model appears in new datasets).
- **Reverse Angle**: If AI training data becomes a public registry (like DNS), this tool dies. But you have at least a 6-12 month window.

#### Signal: "Crawlie – Free open-source SEO audit tool for humans and agents"

- **Plain English**: An open-source SEO audit tool designed for both humans and AI agents. 26 points on HN.
- **Key Judgment**: SEO tools are a crowded market (Ahrefs, Semrush, etc.), but "designed for AI agents" is a differentiator. Watch the category "tools for agents," not the tool itself.
- **Reverse Angle**: If Google changes search algorithms and agent SEO fails, this direction dies.
