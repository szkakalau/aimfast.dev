## 📝 主编说

今天 Hacker News 上最热的讨论是「Are You in the Weights?」——一个让你检查自己模型是否被包含在大模型训练数据集中的工具，241 条评论，457 赞。大家都在聊 AI 训练数据透明度、版权问题、模型泄露。

但真正可构建的信号藏在 GitHub Trending 里：**AI agent "skill" 生态正在爆发**。今天有 3 个超过 9 万 star 的 agent skill 项目同时冲上趋势榜——UI/UX 设计 skill、跨平台内容研究 skill、小红书卡片生成 skill。这些不是模型，而是让 Claude Code、Codex 等 AI coding agent 拥有特定领域能力的插件系统。

谁会先付钱？**用 Claude Code 做 UI 开发的独立开发者**。他们每天花 2-3 小时手动调样式、写设计代码，现在一个 skill 就能让 AI 自动生成。为什么是这周？因为 nextlevelbuilder/ui-ux-pro-max-skill 在 202 天内涨到 94,354 star，增速还在加速——说明这个需求不是小众，而是刚需。

真正麻烦的活：不是写 skill 本身，而是找到那个「AI 做不好、但你付钱就值得」的垂直场景。今天我给你一个。

## 🎯 今日 2 小时构建：UI Skill Studio

**产品名**: UI Skill Studio（UI 技能工坊）
**一句话描述**: 一个让非前端开发者也能生成专业 UI 的 Claude Code skill 市场——今天先做「表单生成器 skill」

**支撑证据**:
- nextlevelbuilder/ui-ux-pro-max-skill 在 GitHub 94,354 star（202 天），今天冲上趋势榜
- 同时有 2 个同类 skill 项目（mvanhorn/last30days-skill、op7418/guizang-social-card-skill）在同一天上榜
- Hacker News 上「Are You in the Weights?」241 条讨论，证明开发者对 AI 生态工具的付费意愿很强

### 为什么不选另外两个方向

1. **「Are You in the Weights?」类产品**：虽然讨论量高（241 条），但这是单次查询工具，没有持续付费场景。用户查一次就知道自己是否在数据集中，然后就没有然后了。定价只能是 $0.99 一次性，天花板太低。

2. **PDF 扫描化工具（Make PDFs look scanned）**：38 条讨论，用户场景太窄——只有需要伪造扫描件的极少数人。而且已有替代品（CamScanner 等），没有差异化空间。

### 定价

- **$19/次**: 定制 skill 生成（你描述需求，我帮你写一个 Claude Code skill 文件）
- **$9/月**: 订阅 skill 市场访问权（每月新增 5-10 个垂直场景 skill）
- **第一版**: Google Form + GitHub Gist 交付，不写一行后端代码

### 最快验证路径（今天就能做）

1. **15 分钟**: 在 GitHub 上 fork nextlevelbuilder/ui-ux-pro-max-skill，把它的 UI 组件库改成「表单生成器」专用版
2. **15 分钟**: 写一个 Twitter/X 帖子：「我写了一个让 Claude Code 自动生成漂亮表单的 skill，想要的人回复 'form'，我免费发」
3. **30 分钟**: 给前 10 个回复的人手动发 skill 文件，问他们是否愿意付 $19 定制更多场景
4. **结果判断**: 如果 24 小时内 10 人中有 3 人愿意付费 → 做产品；如果 < 3 人 → 换个场景重试

**MVP 保持人工**: Google Form + Markdown 文件，不要一上来就做平台

## 🔍 发现机会

### Solo-founder 产品发布

#### 信号: 「Are You in the Weights?」— 模型数据集查询工具

- **白话解读**: 一个网站让你输入模型名称，查询它是否被包含在已知的训练数据集中。241 条讨论说明开发者对「我的模型被谁用了」这件事很焦虑。
- **关键判断**: 这是单次查询工具，没有复购场景。除非做成企业级监控服务（$29/月监控你的模型是否出现在新数据集中），否则不值得构建。
- **反向视角**: 如果 AI 训练数据变成公共注册表（类似 DNS），这个工具就死了。但至少还有 6-12 个月窗口。

#### 信号: 「Crawlie – Free open-source SEO audit tool for humans and agents」

- **白话解读**: 一个开源的 SEO 审计工具，专门为人类和 AI agent 设计。26 分在 HN 上。
- **关键判断**: SEO 工具市场竞争激烈（Ahrefs、Semrush 等），但「专为 AI agent 设计」是差异化点。值得关注的不是工具本身，而是「为 agent 做工具」这个品类。
- **反向视角**: 如果 Google 改变搜索算法导致 agent SEO 失效，这个方向就没了。

### 搜索词暴涨

**今日无显著发现**（搜索趋势数据中只有「agent memory」下降 72%，没有暴涨信号）

### GitHub 快速增长开源项目

#### 信号: omnigent-ai/omnigent

- **白话解读**: 一个开源 AI agent 框架，能统一编排 Claude、GPT、Gemini 等多个模型。26 分在 GitHub Trending。
- **关键判断**: 多模型编排是刚需——开发者不想被锁定在一个模型上。但已经有 LangChain、AutoGen 等成熟方案，Omnigent 需要极致的易用性才能突围。
- **反向视角**: 如果 Anthropic/OpenAI 推出原生多模型支持，这类框架就失去价值。

### 开发者在抱怨什么

#### 信号: 「The Reliability Problem That Forced Us to Rethink AI Agents」

- **白话解读**: 开发者抱怨 AI agent 不可靠——今天能用，明天可能因为模型更新就坏了。
- **关键判断**: 这是 agent 生态的核心痛点。一个「agent 行为测试套件」（类似单元测试但针对 AI agent）可能有机会。定价 $19/月。
- **反向视角**: 如果模型本身变得足够可靠，这个需求会消失。但至少 2-3 年内不会。

## 🏭 竞争情报

### Indie 开发者收入与定价讨论

#### 信号: 「副业做 Chrome 插件 4 个月，收入从 0 到月入 1500 刀」

- **白话解读**: 一个独立开发者分享如何从 0 开始做 Chrome 插件，4 个月达到 $1500/月。
- **关键判断**: 关键不是 Chrome 插件，而是他的推广策略：在相关 Reddit 子版块发帖、在 Product Hunt 发布、在 Twitter 上持续分享。这是可复制的路径。
- **反向视角**: Chrome 插件市场已经拥挤，$1500/月可能是天花板。

### 沉寂老项目突然复活

#### 信号: 「Reviews have become expensive, rewrites have become cheap」（82 赞 / 74 评论）

- **白话解读**: 一个 HN 讨论说「代码审查变贵了，重写变便宜了」——因为 AI 让重写代码成本极低。
- **关键判断**: 这意味着「代码审计」和「代码迁移」工具的需求会上升。如果你做一个「AI 驱动的代码迁移评估工具」——输入旧代码，输出迁移成本估算——可能有机会。定价 $29/次。
- **反向视角**: 如果 AI 写代码质量足够好，根本不需要迁移——直接重写就行。

### 「XX 已死」或迁移文章

#### 信号: 「Ask HN: Due to spam on GitHub, what platforms can I move my projects?」（54 赞 / 55 评论）

- **白话解读**: 开发者因为 GitHub 垃圾信息问题，在询问替代平台。
- **关键判断**: 这不是 GitHub 要死的信号，而是「开发者对平台不满但找不到替代品」的信号。如果你做一个「GitHub 替代品对比指南」PDF，定价 $5，可能有人买。
- **反向视角**: GitHub 不会因为垃圾信息就衰落——它已经太根深蒂固。

## 🎬 行动触发

### 2 小时构建（详细版）

**今天下午 3 点前完成**:

1. **15 分钟**: 在 GitHub 创建 repo「form-skill」，fork ui-ux-pro-max-skill 的代码
2. **30 分钟**: 修改 prompt 模板，让 AI agent 能生成「带验证、带状态管理、带响应式」的表单
3. **15 分钟**: 在 Twitter 发帖：「我写了一个让 Claude Code 自动生成专业表单的 skill。想要的人回复 'form'，我免费发前 10 个」
4. **30 分钟**: 手动给前 10 个回复的人发 skill 文件，附带一个问题：「如果我要做更多场景的 skill（仪表盘、邮件模板、设置页），你愿意付 $19 定制吗？」
5. **30 分钟**: 收集回复，判断是否继续

**完整周末版本**:
- 如果验证通过，做 5 个垂直场景的 skill：表单、仪表盘、邮件模板、设置页、登录页
- 定价 $19/个 或 $49/套
- 发布到 Product Hunt

### 定价和变现模型研究

- **Skill 市场模式**: $9/月订阅，每月新增 5-10 个 skill
- **定制模式**: $19/次，按需生成 skill
- **捆绑模式**: $49 买断 5 个 skill 包
- **推荐模式**: 如果用户推荐朋友购买，双方各得 1 个月免费

**第一版只做定制模式**——Google Form + GitHub Gist 交付，0 开发成本。

### 今天最反直觉的发现

**AI agent 生态的最大机会不是做 agent 本身，而是做 agent 的「技能插件」**。

就像 2016 年 Slack 的最大机会不是做聊天工具（Slack 已经做了），而是做 Slack bot 插件。今天 Claude Code 和 Codex 就是新的 Slack，而「skill」就是新的 bot。
