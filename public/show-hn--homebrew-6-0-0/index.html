How It Works

1

Paste your package list

Open your terminal and run brew list --formula. Copy the output — it’s a simple list of every package Homebrew manages on your machine. Paste it into the field above. No sign‑up, no uploads, no hidden data collection.

2

Get your personalised report

Our engine cross‑references each package against a hand‑curated database of Homebrew 6.0.0 breaking changes, deprecated formulae, and known community issues. You’ll see a colour‑coded compatibility score (green / yellow / red) with direct links to official migration notes and GitHub discussions.

3

Follow the action checklist

Every red or yellow package comes with a step‑by‑step fix: which flag to add, which formula to pin, or which alternative to install. The checklist is ordered by risk, so you can upgrade safely without breaking your daily workflow. Done in under 10 minutes.
