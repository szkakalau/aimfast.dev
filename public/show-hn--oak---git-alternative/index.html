Agent messed up your files? Roll back instantly.

AgentReplay is a Git audit log for AI coding agents. It automatically records every file change, generates human-readable summaries, and lets you roll back to any safe state with one click.

Why Agent Audit Logs Matter Right Now

Every day, thousands of developers use Cursor, Claude Code, Copilot, and other AI coding agents to speed up their work. But there’s a dark side: agents can silently overwrite config files, delete important functions, or introduce subtle bugs that only surface hours later. A recent Hacker News discussion about Oak — a Git alternative for agents — gathered 185 comments, with the top complaints all circling the same pain: “agent commits are meaningless,” “I can’t trace what my agent did,” and “I spent a whole day recovering from an agent’s file change.”

Existing tools like GitLens or raw git diff are built for human commits, not for the chaotic, high-frequency writes of an AI agent. They show you a wall of diffs but no context. You get “fix typo” messages instead of “refactored the user validation module.” And rolling back to a specific agent action? That requires manually hunting through commit logs, stashing, and hoping you don’t lose work. It’s fragile, slow, and stressful.

Now is the perfect time for a purpose-built solution. The signal from Oak’s thread is clear: developers are desperate for better agent oversight. AgentReplay sits on top of your existing Git workflow — zero migration, zero new commands. It’s the “dashcam” for your agent, giving you a clear timeline of every operation, an AI-generated summary of what happened, and a one-click rollback to any point before things went wrong. With the number of agent users doubling every quarter, this pain is only going to grow.

How It Works

1

Point it at your project

Give AgentReplay a local file path or a Git repository URL. It instantly starts monitoring file changes — no configuration files, no daemon setup. Just a single command and you’re ready.

2

AI generates a human-readable timeline

Every time your agent writes a file, AgentReplay groups the changes into operations, then uses a local LLM to translate the diff into plain English. Instead of “update main.py” you see “added rate limiting to the API client.”

3

Roll back with one click

See something suspicious? Each timeline entry has a “roll back to here” button. Click it, and AgentReplay reverts your project to the exact state before that agent operation — no stashing, no manual checkout, no risk.

What You Get

Zero-config monitoring

No need to change your workflow. AgentReplay hooks into your existing Git repository and automatically detects every file write made by your AI coding agent. No plugins, no background services — just a single command to start.

AI-powered summaries

Forget meaningless commit messages. AgentReplay uses a local LLM to analyze every diff and produce a concise, human-readable description of what the agent actually did — like “extracted database connection into a separate module” instead of “update db.py”.

One-click rollback

When an agent changes a file it shouldn’t have, just click “roll back to here.” AgentReplay restores your project to the exact state before that operation — no manual git checkout, no stashing, no risk of losing other changes.
