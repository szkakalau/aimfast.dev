How It Works

1

Describe your agent’s goal

Write one sentence about what you want your agent to do — for example, “summarize today’s top HN stories and email me the digest.” No technical jargon needed.

2

Pick tools & model

Select from a curated list of capabilities: web search, code execution, file I/O, API calls. Then choose your LLM — GPT-4o, Claude 3.5, or a local model. That’s it.

3

Download & run

Get a complete Python script (agent.py), a configuration file (agent_config.json), and a requirements.txt. Run pip install -r requirements.txt && python agent.py — your agent is alive.
