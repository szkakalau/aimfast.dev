Quick answers

What's an MCP server?

Model Context Protocol (MCP) lets AI coding agents talk to external tools — browsers, databases, APIs, file systems. MCPilot manages all of them for you.

Which agents do you support?

Claude Code, Codex, Cursor, GitHub Copilot, Windsurf, OpenCode, AntiGravity, and any agent that uses the MCP standard. We auto-detect your installed agents.

Is my data safe?

MCPilot runs locally. Your MCP configs, API keys, and server data never leave your machine. We can't see them — the dashboard is a local web UI.
